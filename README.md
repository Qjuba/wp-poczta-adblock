# Adblocker for https://poczta.wp.pl
Simple script for skipping annoying popup adblock-warning and removing advertisement divs.
It can be used by greasemonkey extension etc.
This is additional script, because ublock origin doesn't support that type of ads.

Skrypt do pomijania popup'u z ostrzeżeniem o adblocku i usuwania reklamowych div'ów.
Może być użyty w rozszerzeniu greasemonkey itp.
Jest to skrypt pomocniczy, ze względu na to, że ublock origin nie wspiera tego typu reklam.
