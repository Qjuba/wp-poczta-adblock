# wp-poczta-adblock-popup-skip
Simple script for skip annoying popup adblock-warning at https://poczta.wp.pl.
It can be used by greasemonkey extension etc.

Skrypt do pomijania popup'u z ostrzeżeniem o adblocku na stronie https://poczta.wp.pl
Może być użyty w rozszerzeniu greasemonkey itp.
